# shreevarada
A Business Information website for Shreevarada Enterprise Daman

### Requirements
- Node.js version 12.x.x
- Python2 version 2.7.x (for node-sass)
- Python3 version 3.x.x (for flask)

### Installation

```sh
$ git clone
$ cd shreevarada
$ cd npm install
$ cd ..
$ python3 -m flask run
```
